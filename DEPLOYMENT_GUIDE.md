## 🚀 배포 방법 (Netlify 기준)

기존 README.md에 상세 안내가 있습니다. 핵심만 다시 적으면:

### 방법 A: 드래그 앤 드롭 (가장 쉬움 · 1분)
1. https://app.netlify.com/drop 접속
2. zip 폴더 **전체**를 브라우저에 드래그
3. 10~30초 후 자동 URL 발급

### 방법 B: GitHub 연동 (자동 배포)
1. GitHub에 저장소 생성 후 push
2. Netlify에서 "Import from GitHub"
3. 이후 `git push` 할 때마다 자동 배포

> 커스텀 도메인 연결은 기존 README.md 참조

## 🆘 자주 겪는 문제

**Q. 이미지·영상이 안 나와요**  
→ `assets/` 폴더가 함께 업로드되었는지 확인. 상대경로(`assets/xxx.jpg`)여야 함

**Q. 업데이트가 안 보여요**  
→ 브라우저 캐시. `Ctrl+F5` (Windows) / `Cmd+Shift+R` (Mac) 또는 시크릿 창으로 접속

**Q. 로컬에서 `index.html`을 더블클릭했더니 CSS·이미지가 안 나와요**  
→ `file://` 프로토콜 문제. 간단한 서버 필요:
```
cd 해당폴더
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```
