📘 업로드 방법

  1. 위 "전체 복사" 버튼 클릭
  2. GitHub 저장소 → notes.json 파일 열기
  3. 연필 아이콘(Edit) 클릭 → 내용 전체 선택(Ctrl+A) → 붙여넣기
  4. 하단 "Commit changes" 초록 버튼 클릭
  5. 1~2분 후 홈페이지 새로고침 → 반영 완료
